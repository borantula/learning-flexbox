I finally decided to take my time to learn Flexbox. Following the table layouts, 960 fixed grids, responsive grids now it is time for flexbox and css grids (not fully supported yet but coming soon!). So it is perfect time to learn Flexbox!

I will first write all properties and explain briefly. Then I'll try to combine those to create more realistic examples.

I'm preparing this material purely for my own educational purposes, you may find more comprehensive tutorials out there but I'll be glad if this also helps you.

Terminology & Setup

Before diving into the code, let's figure out the terminology first.

  • Container: Parent item that contains flex items. Also called flex container and defined with display:flex
  • Item: items in a flex container, also called flex items.

Example:

Here we have boxes within a 100% wide container. Boxes (flex-item's) doesn't have width and height but only min-width min-height

Alignment

                    


                    

                        

                            Aligning items with properties like float, text-align, display inline, inline-block, table
                            etc. is
                            always a pain in the c*de and almost always causes some problems.
                        

                        
Well no more with flexbox!

                    


                    

                        
justify-content:

                        

                            This is a property for flex container. Below you can see examples for single and multiple
                            items in a container
                        

                        
                            
                                .flex-container {
                                    display: flex;
                                    justify-content: flex-start | flex-end | center | space-around | space-between;
                                }
                            
                        



                        


                            justify-content:
                            
                        

                        

                            

                            
                        



                        
    
                            
  • flex-start: items are packed at the start of container
    • 
                            
  • flex-end: items are packed at the end of container
    • 
                            
  • center: This one is Captain Obvious. Items are packed at the center of
                                the container. This
                            
    • 
                            
  • space-between: items are evenly distributed after placing first and
                                list element without any space on both ends. If there is only one item it acts like
                                flex-start. Good for menu items.
                            
    • 
                            
  • space-around: items are evenly distributed with equal space on both
                                sides.
                            
    • 

                        



                    

                    



                        
align-items:

                        

                            This is a property for flex container. It is cross axis version of justify-content.
                        

                        
                            
                                .flex-container {
                                    display: flex;
                                    align-items: flex-start | flex-end | center | baseline | stretch;
                                }
                            
                        



                        



                            align-items:
                            
                        

                        

                            

                            
                        


                        
    
                            
  • flex-start: items are packed at the start of container
    • 
                            
  • flex-end: items are packed at the end of container
    • 
                            
  • center: This one is Captain Obvious. Items are packed at the center of
                                the container. This
                            
    • 
                            
  • baseline: flex item baselines aligned with cross axis baseline.
                            
    • 
                            
  • stretch: if the dimensions on item are auto, it stretches the item on
                                cross axis. It respects the min-width/min-height/max-width/max-height properties.
